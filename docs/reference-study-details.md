## 實際觀察

首頁首次載入先以深色影像、極大英文標題及極少導覽建立張力；固定 Header 讓企業、方法、事例、採用、知見維持可達。內容往下轉為作品索引、方法、知見與頁尾，結構並非單純大圖，而是「強首屏—高密度案例—克制文字」的節奏。作品索引以類別、名稱與一句說明形成可掃描組合，詳頁再以大圖、段落錨點、概要、背景、目標、支援內容、成果與 Credits 組成長閱讀。

Hover 主要透過圖像與文字狀態變化告知可點；圖片採延後載入，首屏與詳頁會有短暫黑場／淡入。桌面保留強烈影像裁切及固定導覽；手機必須降低裁切並維持完整題名。其動效依賴影像與文字揭示，視覺刺激較高。

## 可轉化原理

- 精準字級比例與固定 Header，用於本站的學術索引與閱讀定位。
- 「索引—詳讀」兩層架構，用於期刊、研討會與 CV。
- 深色模式只保留黑白對比及單一品牌色，不移植背景影像或材質。
- 作品詳頁的段落錨點轉化為 CV Sticky Index 與論文穩定 Anchor。

## 刻意不採用

不採用全屏影像、顆粒影片、長黑場、字形拆分動畫、品牌案例圖、固定 Contact 抽屜及其程式碼。學術網站首頁需要立即辨識姓名、身分、研究與成果，不能讓影像先於內容。
