## 導入原則

1. 採取「資料先於介面」：原始題名、日期、書目與連結始終可讀、可複製、可列印。
2. 採取「互動可降級」：Canvas、視覺化與動態均有文字與靜態資料替代。
3. 採取「編輯排印而非模板卡片」：以欄線、索引、字級、留白與資料層次構成版面。
4. 不使用任何外部案例的圖片、內容、品牌或未授權視覺資產。
