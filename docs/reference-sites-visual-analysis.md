# Details 與 JUNNI 實證視覺分析

稽核日期：2026-07-18。以桌面瀏覽器實際開啟首頁、導覽、作品索引、內容段落與 Footer，並檢查可見 DOM、互動節點及不同內容密度。

## Studio Details

- 首屏以極少文字建立尺度對比；超大主標與小型日文敘述分離，不落在同一中央軸線。
- 導覽將每個字元包成獨立運動單位，原字串仍由螢幕閱讀器完整讀取。
- Projects 使用大型影像／題名與小型類別資料，不以等寬卡片陳列。
- 長頁面在 Projects、Approach、Knowledge、Careers、Contact 間切換密度與背景，Footer 本身是一個完整資訊場景。
- 可借鑑：精密對齊、超大與微型字體並置、字元級回饋、長頁面節奏。
- 不適合直接移植：商業品牌的影像主導與聯絡轉換強度；學術站須讓題名、卷期與年份保持第一閱讀優先。

## JUNNI

- 首頁以寬字距、重複文字、字元拆分與大幅區塊移動形成節奏。
- Works、Service 與 Recruit 不採同一版型；大型英文字成為段落轉場與方向標記。
- 連結文字使用雙層字串切換，重複跑馬形成低速持續運動；Footer 延續整頁互動語言。
- 可借鑑：全畫面導覽、字元拆分、水平文字、不同章節的方向變化、桌機與手機不同的回饋。
- 不適合直接移植：高刺激 WebGL、娛樂性移動與商業作品預覽；本站將幅度、速度與同頁數量降低。

## 轉化原則

新站不複製品牌、Logo、圖像、程式碼或完整構圖。Details 的尺度紀律轉為學術索引；JUNNI 的動態密度轉為低幅度文字、紙面與欄線運動；MUJI 的「空」則透過非對稱留白、自然材料色與克制互動統合三者。
