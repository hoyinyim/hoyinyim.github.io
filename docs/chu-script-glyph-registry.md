# 戰國楚系文字候選登錄

本輪登錄 8 個可追溯候選：4 個「心」字形與 4 個「文」字形。每筆均記錄小學堂字號、出處簡號／篇號、《楚系簡帛文字編（增訂本）》索引及 CC0 1.0 授權。機器可讀資料在 `src/data/chu-script-glyphs.json`。

這些只屬候選，不等於已核准 Logo；`userApproved` 與 `productionEnabled` 均為 `false`。網站現有古文字仍依原始來源標記為甲骨文，未冒稱楚系文字。

8 個資料庫原始字形圖的逐筆連結位於 `docs/chu-script-logo-research.md`；連結只供使用者核定，正式網站不會熱連小學堂圖像。
