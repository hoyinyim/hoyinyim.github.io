# 古文字動態 Menu 設計

參考研究網址：`https://prj-foodecon.w.waseda.jp/`。本站只轉化其線性構件與全幕展開邏輯，不複製原圖。

## 關閉狀態

「選單」按鈕保留清楚 Accessible Name。按鈕右側以核定「學」字資產的水平遮罩片段與三道線性構件形成壓縮記號；Hover 僅作 1–2px 位移與數度偏轉，不旋轉一圈、不發光、不追隨游標。

## 開啟狀態

1. 線性構件回應點擊。
2. 全螢幕底板由右向左展開，620ms。
3. 完整古文字由按鈕方向進入左側圓形舞台，680ms。
4. 六項繁中導覽以 45ms 間距依序出現，第一項 150ms 後即可操作。
5. 搜尋、模式與 Email 於 420ms 後出現。

總長不超過 680ms。關閉則在 410ms 內反向收束，焦點回到「選單」按鈕，背景鎖定解除。

## 版面

桌面為左側古文字舞台、右側六項主導覽與四項著作子類；手機將舞台縮至上方，導覽維持現代繁中文字並可完整捲動。古文字是 `aria-hidden` 的視覺層，不取代連結文字。

## 技術

使用原生 `dialog`、CSS Grid、CSS Custom Properties、SVG 與少量 JavaScript。沒有 Canvas、WebGL、Three.js、GIF、影片 Menu 或大型動畫框架。
