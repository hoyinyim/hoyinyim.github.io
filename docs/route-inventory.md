# 路由清單

以下十個主要路由與一個錯誤路由均保留於 GitHub Pages 根目錄；所有導覽消費同一份 `src/data/routes.json`。

| ID | 路徑 | 中文標籤 | 英文標籤 | 類型 |
| --- | --- | --- | --- | --- |
| home | `index.html` | 首頁 | Home | 主要頁面 |
| about | `about.html` | 關於 | About | 主要頁面 |
| research | `research.html` | 研究 | Research | 主要頁面 |
| journal | `journal-papers.html` | 期刊論文 | Journal Articles | 成果檔案 |
| conference | `conference-papers.html` | 研討會論文 | Conference Papers | 成果檔案 |
| translations | `translations.html` | 譯著／哲學普及作品 | Translations | 成果檔案 |
| certificates | `certificates.html` | 證照／證書／獎項 | Credentials | 學術檔案 |
| teaching | `teaching.html` | 教學 | Teaching | 精簡帳冊 |
| cv | `cv.html` | 履歷 | CV | 可列印彙整 |
| contact | `contact.html` | 聯絡 | Contact | 聯絡場景 |
| not-found | `404.html` | 找不到頁面 | Not Found | 錯誤頁面 |

## 路由消費者

Desktop Header、全螢幕 Menu、搜尋快速入口、Footer、Breadcrumb、Sitemap 與當前頁標記不得另行手寫路由。
