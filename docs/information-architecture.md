# 資訊架構

## 既有資料對應路由

| 路由 | 可用既有資料 | 原始來源 |
| --- | --- | --- |
| `/` | 姓名、身分、首頁介紹、研究標籤、學歷、教學與學術經驗 | `index.html` |
| `/about.html` | 個人資料、個人介紹、學歷、教學與學術經驗、照片 | `index.html` |
| `/research.html` | 已有研究標籤、目前研究方向、研究專長 | `index.html` |
| `/publications.html` | 既有期刊論文資料 | `journal-papers.html` |
| `/talks.html` | 既有研討會資料 | `conference-papers.html` |
| `/translations.html` | 譯著與哲學普及作品 | `translations.html` |
| `/credentials.html` | 證照、證書與獎項 | `certificates.html` |
| `/cv.html` | 已有學歷與教學／學術經驗 | `index.html` |
| `/contact.html` | 現有電子郵件與可驗證既有外部連結 | 各既有頁面 |

## 不建立的虛構資料頁

現有來源未提供可獨立遷移的研究計畫、完整課程清單、札記、ORCID、Google Scholar、GitHub 個人頁或完整 CV。重構期間不會為了補足導覽而生成該等內容；僅在原有資料存在時新增對應入口。
