# 網格與比例系統

## 桌機

- 16 欄等分網格，頁面左右安全距離 `clamp(24px, 3vw, 48px)`。
- 欄間距 `clamp(12px, 1.25vw, 24px)`；內容最大寬度 1680px。
- 開場區至少使用兩條清楚軸線：標題軸與資料軸。
- 長文建議佔 7–9 欄；索引或 Metadata 佔 3–4 欄；其餘留白保留節奏。

## 平板與手機

- 900px 以下收斂為 8 欄；700px 以下固定為 4 欄。
- 手機左右安全距離 18px、欄間距 8px。
- 任何欄位都必須設定 `min-width: 0`，長網址以 `overflow-wrap: anywhere` 處理。
- 正式題名自然換行；不使用省略號裁切，不以固定高度限制內容。

## 垂直節奏

- 基準單位 8px。
- 小間距 8／12／16px；段落 24／32px；章節 64／96／144px。
- 列表分隔線與資料對齊優先於卡片留白，手機版不保留桌機的空白欄。
