# 戰國楚系文字姓名標誌研究

查核日期：2026-07-21。

## 結論（2026-07-21 更新）

本輪仍不製作「嚴浩然」楚系文字姓名標誌，也不把既有甲骨文字形改稱楚系文字。三字尚未同時具備可核對記錄與使用者核准，因此正式網站繼續以真正 HTML 文字「嚴浩然」作 Wordmark。

另已從小學堂登錄 8 個「心／文」楚系候選。小學堂授權頁明示查詢結果字形圖與字形屬性採 CC0 1.0，故「授權未確認」不再是這 8 筆候選的問題；目前唯一尚未通過的是使用者對具體字形的視覺核准。詳見 `src/data/chu-script-glyphs.json` 與 `docs/chu-script-glyph-registry.md`。

## 已查範圍

- 香港中文大學「漢語多功能字庫」的「嚴」「浩」字頭頁。
- 中央研究院「小學堂文字學資料庫」及其技術報告所述楚系簡帛資料架構。
- 《戰國文字字形表》《楚系簡帛文字編》的公開書目資料與研究說明。

上述資料能證明戰國文字必須區分楚、秦、齊、燕、晉等區系，也能提供進一步紙本覆核方向；但公開搜尋結果仍不足以把三個現代姓名字直接定為可用楚系 Logo 字形。沒有以一般篆書、AI 生成、字體套打、鏡像、旋轉或拼造補足缺字。

## 後續核定程序

若日後要採用，須逐字回到《楚系簡帛文字編》或可靠數位記錄，記下出土材料、簡號、照片／拓片頁、釋讀依據、年代與授權。三字齊備後仍須由使用者核准候選；若只找到部分字，不得補造，而應另行研究單一「文／學／言／心／身」等候選 Symbol，並再次取得核准。
