# 元件系統

## 全站元件

- **Site Header**：品牌、四個核心入口、Search、Theme、Menu。手機保留品牌、Theme 與 Menu。
- **Grouped Menu Dialog**：PROFILE、RESEARCH & WORK、PRACTICE 三組；焦點鎖定、Escape 關閉、還原觸發點。
- **Search Dialog**：搜尋頁名與內容索引；初始 HTML 仍保留所有可閱讀內容。
- **Site Footer**：只出現一次，包含正式身分、Email 與頁面索引。
- **Metadata Rail**：年份、卷期、狀態、類型與標籤使用一致字級與行距。

## 內容元件

- **Journal Entry**：出處 → 題名 → 卷期頁碼 → 狀態／索引 → 全文與複製引用。
- **Conference Published Strip**：已出版論文採橫向編輯條帶；其餘發表採年代帳冊。
- **Research Focus**：三個研究焦點各有獨立欄位比例；專長以矩陣收束。
- **Credential Ledger**：證照、證書與獎項分流；沒有真實圖像即只呈現資料，不做假預覽。
- **Education Timeline**：可捲動、無分頁、無頁籤；時間、學位、機構固定對齊。
- **CV Index**：桌機黏附索引、A4 列印版與網站資料共源。

## 元件驗收

所有互動元件須有可見焦點、44px 觸控尺寸、正確名稱與狀態；所有內容元件須在 320px、200% 文字縮放、關閉 JavaScript與列印時維持可讀。
