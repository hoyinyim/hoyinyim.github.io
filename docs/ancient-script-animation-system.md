# 古文字動態系統

## 動態類型

1. 首次進場：A 級字形以低幅透明度與裁切顯示，避免大距離飛入。
2. 區段進場：B 級字形由 Intersection Observer 加上顯示狀態，只進行一次。
3. Hover／Focus：C 級索引提高透明度，相關細線延展；鍵盤 Focus 與 Hover 使用相同回饋。
4. 頁面轉換：站內連結啟動 220ms 單色字形轉場，再前往新頁。
5. 聯絡回應：複製電郵後，「言」只作短暫回應，不取代成功狀態文字。

## 限制

動態不旋轉整頁、不使用視差追蹤、不連續循環、不在捲動事件逐幀修改字形。單次轉場不超過 `--glyph-motion-slow`，位移保持 1–4px 或小比例裁切。

## Reduced Motion

`prefers-reduced-motion: reduce` 下，所有古文字動畫、轉換、位移與頁面轉場均停用；B 級字形直接呈現固定低透明度。功能、內容順序與可操作項目完全不變。
