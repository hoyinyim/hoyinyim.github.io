# 第二階段視覺與內容驗收

## 驗收範圍

- 首頁、關於、研究、履歷、聯絡、期刊論文、翻譯作品及其共用導覽與頁尾。
- 桌面主要基準為 1280 × 720；手機基準以 390px CSS 斷點與可點擊尺寸進行檢查。
- 內容完整性由原始快照雜湊、頁面結構與既有自動測試共同核對。

## 已完成檢查

| 項目 | 結果 | 備註 |
| --- | --- | --- |
| 首頁首屏比例 | 通過 | 姓名、身份與導覽在 1280px 畫面內形成非置中的左右構圖，沒有莫名斷行。 |
| 期刊論文列表 | 通過 | 瀏覽器實測 10 筆項目；標題、年份、索引與狀態欄維持可讀。 |
| 水平溢出 | 通過 | 主要頁面 `scrollWidth` 不大於可視寬度；390px 規則移除固定欄寬與橫向裝飾。 |
| 字級與行長 | 通過 | 長題名設最大字級；正文維持 16–18px 與 1.75–1.9 行高；資料標籤不承擔正文。 |
| 導覽 | 通過 | 固定 MENU 可開啟全幕導覽；命令面板與全幕導覽分離，不再互相觸發。 |
| 鍵盤與動態 | 通過 | 導覽具 Esc 關閉與焦點回復；Reduced Motion 停用載入、遮幕與顯影位移。 |
| 圖片替代文字 | 通過 | 已檢查主要內容圖，沒有空白 `alt`。 |
| 內容完整性 | 通過 | 自動測試核對原始內容快照、頁面標題與必要內容；未改寫個人或學術敘述。 |
| JavaScript | 通過 | 語法檢查與站點建置通過。 |

## 實際瀏覽器觀察

- 首頁：暖紙色背景、姓名主標、右側身份資料、直排英文名與細線索引在首屏內穩定顯示。
- 期刊論文：舊卡片牆已改為編輯式大列表；年份與索引可以快速掃讀，列表不以裝飾陰影製造層級。
- 關於與研究：肖像、段落、研究分類使用不同欄位節奏，不再套用首頁的構圖。
- 行動版：選單、頁面標題、內容欄與頁尾改為單欄；觸控環境停用自訂游標，按鈕維持可操作尺寸。

## 限制與保留

- 本輪驗收以靜態截圖、即時 DOM 尺寸、互動操作與自動測試為準。現有瀏覽器控制環境未提供錄影匯出，因此沒有偽造影片檔；動態已逐項以實際操作驗證。
- 不同裝置的中文字型會依系統可用字型回退；版面以字級上限、最小欄寬與換行規則避免因此破版。
