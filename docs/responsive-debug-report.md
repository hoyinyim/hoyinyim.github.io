# 全站響應式除錯報告

測試日期：2026-07-19。測試對象：第三階段本機版本。

## 結論

Chrome 自動測試共完成 120 個「頁面 × 視窗寬度」組合，最終結果為 0 項失敗。另通過桌面與手機全螢幕選單、首頁 CTA、兩筆指定外部連結、125％至 200％縮放等效視窗、Reduced Motion 及 Google Fonts 阻擋後的系統回退字型測試。

## 測試矩陣

- 寬度：320、360、375、390、430、540、768、820、1024、1280、1440、1920px。
- 頁面：首頁、關於、研究、期刊論文、研討會論文、譯著／哲學普及作品、證照／證書／獎項、教學、履歷、聯絡。
- 縮放：以 1280px 實體視窗換算 125％、150％、175％、200％的有效版面寬度，確認真正會觸發對應 Media Query 的版面狀態。
- 動作：展開／關閉 Menu、Escape 關閉、焦點移入、當前頁標示、首頁 CTA 導頁、典藏預覽初始化。

## 每個組合的自動檢查

1. 文件寬度不得超出視窗。
2. 正文基準字級須為 17–19px。
3. 必要 Metadata 不得小於 14px。
4. 顯示中的桌面主導覽不得小於 16px。
5. 長題名與正文不得被 `hidden` 或 `clip` 裁切。
6. 固定操作元件不得超出視窗。

## 首輪失敗與修正

| 問題 | 首輪數量 | 修正 |
| --- | ---: | --- |
| 固定元件誤報 | 96 | 排除依無障礙慣例在未聚焦時收於視窗外的 Skip Link；其他固定按鈕仍完整檢查 |
| Metadata 小於 14px | 48 | 提高論文年份、側欄索引、標籤、譯著來源與必要控制文字的選擇器優先級 |
| 水平溢位 | 11 | 修正證照典藏工作區沿用舊 Grid line 的隱性欄位；限制譯著跑馬區與圖片容器的可見邊界 |

修正後測試結果：120／120 組合通過，0 項水平溢位、0 項文字裁切、0 項字級違規。

## 瀏覽器覆蓋

- 已執行：Windows 安裝版 Google Chrome（Chromium），Playwright Core 1.61.1。
- Firefox：本機沒有可用 Firefox 執行檔，未假稱已測。
- WebKit：本機沒有可用 WebKit 執行檔，未假稱已測。
- Codex 內建瀏覽器與 Chrome 外掛均因安全規則拒絕 `127.0.0.1`／本機檔案網址；因此本機檢查由專案內 Playwright 測試執行，公開部署後另作瀏覽器肉眼核對。

## 可重現方式

1. 在儲存庫根目錄執行 `pnpm serve`，預設使用 `http://127.0.0.1:4173`。
2. 執行 `pnpm install`。
3. 執行 `pnpm test:visual`。

機器可讀結果：`docs/qa/third-stage/visual-responsive-results.json`。改造後截圖：`docs/qa/third-stage/after/`。
