# 古文字 Menu 動畫分件圖

參考研究網址：`https://prj-foodecon.w.waseda.jp/`。

## 資產完整性

六個本地 SVG 均完整保存來源資產的 `viewBox` 與 path，不重畫、不簡化、不改變字形結構。正式動畫不切改 SVG path；只控制「完整字形圖層、遮罩顯示、舞台結構軸線」三種顯示分件，因此不存在為動畫任意切斷原字形的問題。

| 分件 | 關閉位置 | 開啟位置 | 方向／時間 | Reduced Motion |
| --- | --- | --- | --- | --- |
| 按鈕字形遮罩 | 右上角 27×23px，水平窄帶 | 不移動；由舞台完整字形接續 | Hover 1–2px／220ms | 靜止 |
| 全幕底板 | 右緣 `scaleX(0)` | `scaleX(1)` | 由右向左／620ms | 立即顯示 |
| 完整古文字 | 控制區方向、縮放 0.08 | 左側圓形舞台中央 | 平移＋縮放／680ms | 靜態完整字形 |
| 結構軸線 | `scaleY(0)` | 舞台中央 76％ 高度 | 向下延伸／480ms | 靜態 |
| 導覽項目 | 右移 24px、透明 | 原位、不透明 | 45ms 錯開／420ms | 立即顯示 |
| 關閉收束 | 開啟位置 | 返回控制區方向 | 180–410ms | 立即關閉 |

聚焦導覽項目時，舊字形淡出、新字形以遮罩完成顯示；兩字之間不作 Morph、不互相融化。古文字載入失敗時只隱藏視覺圖層，六項現代繁中連結保持可用。
