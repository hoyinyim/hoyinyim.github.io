# Design System

「文本宇宙」以 CSS Design Tokens 管理紙白、深墨、朱砂、青銅綠與深靛。亮色模式偏向紙張與印刷；暗色模式改用文獻庫房般的低照度色階，而非單純反相。

個人識別為三條不同高度的索引欄線，從資料欄、校記與排印比例推導；沒有使用印章、書法、竹簡、山水或無來源圖像。資料頁以條列、索引、時間線與欄線為主，避免通用圓角卡片。

第二階段採十六欄流體編輯網格，桌面外距為 `clamp(32px, 5vw, 112px)`。首頁、肖像、研究索引、書目、CV 與 Footer 使用不同跨欄比例；空白欄具有方向，不再以單一 `max-width` 置中全部內容。

動態值統一為快速 220ms、中型 520ms、慢速 880ms、場景 1400ms；緩動以 Expo Out 與 Quart In/Out 為主。所有場景在 Reduced Motion 下直接呈現完整內容。
