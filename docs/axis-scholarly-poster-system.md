# AXIS Scholarly Poster System

## 定位

「軸線式當代學術平面系統」以學術資料的可核對性為核心，採用展覽海報的比例張力、期刊目錄的資訊密度與瑞士式網格的秩序。它不是 AXIS Font 的仿製品，也不使用未授權字型；「AXIS」只描述頁面的視覺軸線與閱讀動線。

## 五項原則

1. **學術內容先於裝飾**：題名、出處、卷期、頁碼、狀態與連結形成固定層級。
2. **大比例只留給身分與章節**：姓名或頁名可以形成海報尺度；正式書目維持正常閱讀比例。
3. **一頁一種開場**：首頁、研究、期刊、關於與選單各自有構圖，不再套用同一個 Hero。
4. **軸線而非卡片**：以直線、欄線、留白與對齊建立區隔，避免大量浮動卡片、圓角與陰影。
5. **互動是索引工具**：Menu、Search、Theme、Filter 與 Citation Copy 都要可用鍵盤、可讀狀態、無動畫亦完整。

## 視覺語法

- 16 欄桌機網格；4 欄手機網格。
- 中文題名採無襯線主導，長篇正文可使用系統襯線字族。
- 淺色為冷白紙面，深色為近黑墨面；全站只有一個青灰藍強調色。
- 細線表示資料關係，粗線只用於開場軸線與目前狀態。
- 數字、年份、卷期與英文小標使用等寬或窄體系統字族。

## 禁用語法

- 未授權的 AXIS Font 或其他商業字型。
- 為了「設計感」切斷中文詞組、縮小正文或省略書目欄位。
- 每頁重複巨型姓名、背景紋理、假古籍裝飾、玻璃擬態與無意義動畫。
- 把沒有圖像來源的證書偽裝成文件縮圖。

## 原型與正式元件對應

五個原型分別驗證：首頁身分海報、研究焦點編排、期刊帳冊、肖像與教育時間線、分組式全站選單。正式頁面只承接其比例、層級與互動規則，內容始終由 `src/data/` 生成。
