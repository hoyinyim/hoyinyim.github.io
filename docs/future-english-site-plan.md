# 未來英文版規劃

英文版將以獨立 `/en/` Locale Route 建立，不與繁體中文全文混排。每個中文路由將有明確的一對一英文對應；未有完整人工英文內容的頁面不發布。

## 預留資料

- 現有路由保留 `locale` 與 `futureLocaleHref` 欄位。
- 未來 Language Switch 只在當前頁存在已發布英文對應時出現。
- 切換語言時保留所在頁面；缺少對應頁時不顯示失效入口。
- 英文內容須人工核定正式姓名、學位、機構、書目與術語，不採機器翻譯自動上線。

古文字 Menu 與語言切換各自獨立；古文字不代表任何語言選項。參考 Menu 研究網址：`https://prj-foodecon.w.waseda.jp/`。
