# 參考站到本站樣式 Token 對照

| 參考特徵 | 本站 Token／規則 | 移植邊界 |
| --- | --- | --- |
| HASHI 窄欄、低字重、嚴格分行 | `--measure-reading`、`line-break: strict` | 不移植日文直排及人工分行 |
| KAITAKSHA 大標與舒展段距 | `--text-display-*`、`--space-6` 至 `--space-9` | 不移植商用字體及過寬字距 |
| AA 800px 內容欄、2 倍行高 | `--measure-long`、閱讀型行高 | 書目列收斂至 1.5，避免頁長失控 |
