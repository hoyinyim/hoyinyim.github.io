# 古文字逐頁構圖規格

## 首頁

「學」位於右側身份資訊後方，與主姓名形成不對稱平衡。三域拓樸分別嵌入「學／言／教」，下方三欄再以微型字形作索引。人物段只在標題附近使用「人」，不進入肖像範圍。

## 關於與研究

關於頁以「人」建立首屏留白，以「行」接續教育時間軸。研究頁不是三張同形卡片，而是三個不同方向的橫向區段：「學」靠右形成研究起點，「冊」在左側穿入出土文獻段，「言」由下方托起荀子研究段；最後專長矩陣回到小型「學」。

## 著作系統

著作總覽使用巨幅淡色「冊」與三條大型橫向分類列；每列同時提供編號、現代中文分類名、數量與簡述。期刊頁把「冊」轉為年份和篩選索引；研討會頁以「言」作首屏主字形，並以「冊」和「行」清楚分隔出版論文集與會議發表。

## 譯著、證照與教學

譯著頁以「言」建立公共表達軸，「冊」只標示成冊譯著。證照頁用「行」表達進程，以「冊」標示資格紀錄，避免獎章與印章隱喻。教學頁以「教」為主，並以「學／行」分開學習內容與實踐路徑。

## 履歷與聯絡

履歷頁主字形透明度低於其他頁，列印時全部隱藏。聯絡頁讓「言」與複製電郵後的短暫線性回應相連，但電郵地址、複製按鈕與狀態文字均保持現代文字。

## 共通邊界

桌機可使用 A／B／C 三級；手機只保留單一 A 級和必要 C 級。所有頁面在 768px 以下隱藏大型 B 級字形，使頁面密度、分行與觸控目標優先於裝飾。
