# 楚系文字語義對應

| 字形 | 現代字 | 限定用途 |
| --- | --- | --- |
| `chu-person` | 人 | 個人介紹 |
| `chu-study` | 學 | 研究、首頁 |
| `chu-writing` | 文 | 期刊論文 |
| `chu-register` | 冊 | 著作總覽 |
| `chu-meeting` | 會 | 研討會論文 |
| `chu-translation` | 譯 | 譯著／公共寫作 |
| `chu-notice` | 告 | 證照、證書與獎項 |
| `chu-teaching` | 教 | 教學 |
| `chu-cv` | 履 | 履歷 |
| `chu-contact` | 聯 | 聯絡 |

路由只可引用核定對應；同一頁內可使用小型輔助字形，但不得取代現代繁體中文標題與導覽文字。
