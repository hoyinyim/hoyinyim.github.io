# 元件尺度矩陣

| 元件 | 桌面尺度／行為 | 平板尺度／行為 | 手機尺度／行為 | 人工判斷 |
| --- | --- | --- | --- | --- |
| Header | 72px 高；姓名、主導覽與三個 44px 控制分層。 | 保留橫向導覽至 760px。 | 64px；主導覽隱藏、保留主題與 Menu。 | Header 不高於內容首屏，也不縮成工具列。 |
| 品牌與小字 | 品牌英文、操作與 email 最低 16px。 | 同。 | 同；最窄 360px 隱藏主題切換以保留 Menu。 | 可操作資訊不以低對比小字犧牲。 |
| 首頁 H1 | 5–9.5rem，身份資料在同屏右側。 | 格線收束。 | 4.5–6.4rem；兩行姓名，非硬擠單行。 | H1 第一焦點，但不搶走人物與研究資訊。 |
| 內容頁 H1 | 一般 3.2–6.2rem；長題名在譯著／證照頁自然換行。 | 依欄位縮放。 | 長題名使用頁面專用範圍，避免單字孤行。 | 四字與八字以上標題不強制同一視覺重量。 |
| Hero | 首頁最大 44rem；資料頁 25–35rem。 | 降低上下 padding。 | 無固定最小高度。 | 留白有節奏，不形成空洞。 |
| 人物照 | 原始比例、`object-fit: contain`。 | 欄位縮窄但未裁切。 | 先顯示完整照片，再接簡介。 | 頭部、臉部與文字距離自然。 |
| 教育校徽 | 成大 62×62px；臺北市立大學 108×62px。 | 同比例縮放。 | 48px 高／橫式 84px 寬。 | 校徽以光學重量平衡，非強制方形。 |
| 古文字 | 主字形 12–32rem、0.06 透明度；著作頁最大 33rem。 | 退為背景。 | 次要區段字形隱藏，主字形低對比保留。 | 僅作第二／第三焦點，不遮文字。 |
| 三域拓撲 | 520–720px 高；三張卡 400px。 | 圓形縮放、保持關係。 | Canvas 430–560px，卡片單欄。 | 首頁兩個 viewport 內出現簡介及拓撲入口。 |
| 書目／年表 | 年份欄、題名、出處、操作有清楚列距。 | 格線重配。 | 線性列序、題名自然換行。 | 長資料頁不成灰色文字牆。 |
| CTA／按鈕 | 最小 44px，主要 CTA 具線與箭頭。 | 同。 | 同，避免手指誤觸。 | 視覺權重低於 H1、高於 Metadata。 |
| Menu | 圖形舞台約 414px；主項約 72px。 | 圖與字同步縮小。 | 圖形約 190px；主項 39px；完整一屏。 | 文字主導，選項數字與 glyph 為輔。 |
| Footer | 內容下直接收束；email 至少 16px。 | 多欄保留。 | 單欄／兩欄資訊重排。 | 不前置巨大空白，結尾清楚。 |
