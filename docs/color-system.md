# 色彩系統

## 核心色票

| Token | 淺色模式 | 深色模式 | 用途 |
| --- | --- | --- | --- |
| `--paper` | `#f4f5f3` | `#0e1110` | 主背景 |
| `--ink` | `#101311` | `#f2f4f1` | 正文與主標 |
| `--muted` | `#59605d` | `#aab2ae` | Metadata 與次要說明 |
| `--line` | `#c8ceca` | `#39403c` | 分隔線與控制項邊界 |
| `--accent` | `#315c64` | `#9bc3ca` | 唯一強調色、目前狀態、連結焦點 |

## 規則

- 色彩不可是狀態的唯一訊號；文字、圖示或 `aria-pressed` 必須同步表達。
- 正文與背景以 WCAG AA 對比為最低門檻。
- `--accent` 不作大面積背景，避免破壞長篇閱讀；全站不新增第二組裝飾色。
- 圖片不套用改變內容辨識的濾鏡；只允許輕微明度調整以融入明暗模式。
