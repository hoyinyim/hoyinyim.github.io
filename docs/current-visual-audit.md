# 第二階段現況視覺稽核

稽核日期：2026-07-18。範圍為正式網站首頁、About、Research、期刊論文、研討會論文、譯著、證照、Teaching、CV、Contact 與 404。

## 主要問題

- 首頁仍以單一深色 Hero、等距章節與規則資料格構成，視覺速度過於平均。
- `field-item`、`work-card`、`public-card` 等卡片型元件仍佔主要版面；譯著頁尤其明顯。
- 一般內容頁共用相同深色 Hero，About、Research、CV 與 Contact 缺乏獨立構圖。
- 既有動畫以淡入、上移及 Hover 變色為主；沒有統一載入、頁面轉場、全畫面導覽與游標語言。
- 首頁 Footer 只是兩段文字，無法形成頁面結束場景。
- 手機雖無橫向溢出，但主要是桌面網格單欄化；空間節奏與觸控回饋不足。

## 既有資產

- 字體：Noto Serif TC、Noto Sans TC、Cormorant Garamond，以及系統後備字體。
- 色彩：紙米色、深墨、靛藍、青銅、苔綠、朱砂。
- 材質：CSS 紙纖維與 Canvas 低密度文字／欄線場景。
- 圖片：個人肖像與譯著封面；不使用無來源圖像。

## 重製判準

保留全部原始內容與順序，以 CSS 網格、方向性留白、尺度對比及原生 JavaScript 互動重做觀看方式。卡片改為索引、跨欄清單、材料桌與大型編輯式段落。
