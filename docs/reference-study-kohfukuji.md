# 興福寺參考研究

## 查核範圍

- 首頁：`https://www.kohfukuji.com/`
- 寺寶・文化財：`https://www.kohfukuji.com/property/`
- 證據：`docs/qa/axis-reference-study/kohfukuji/`

## 實際觀察

首頁以建築大圖、垂直日文導覽、寺名與語言選擇建立文化重量；首屏以下立即提供「本日のご拝観」、公告、基本資訊、特別公開、講座等實用資料。內容不是單純氛圍展示，而是以明確時間、電話、開放狀態與詳情連結服務訪客。寺寶頁以簡短導言及內容入口收束，留白與層級安靜。

桌面以垂直字、巨大建築影像及大量留白形成莊重；手機改用 Menu 入口並重新安排，不維持同樣裁切。微互動主要是選單、連結和狀態提示，速度緩和。

## 可轉化原理

- 文化重量轉化為穩定留白、沉靜速度、明確事實與克制顏色。
- About 肖像以自然色、姓名與身分直接開場；不加濾鏡或裝飾頁碼。
- 公告與講座的時間次序轉化為 Conference Chronology Ledger。
- Menu 的層級觀念轉化為 PROFILE、RESEARCH & WORK、PRACTICE 三組。

## 刻意不採用

不採用寺院建築、佛像、佛教符號、書法 Logo、垂直日文導覽、紅色圓形導覽或宗教裝飾。本站的文化重量必須由學術資料本身產生。

## 版權與反抄襲

未下載或使用興福寺照片、文字、商標或圖示。最終網站只轉化「清楚事實＋留白＋低刺激互動」原則。
